Telemtry Link Information

The host address used for telemetry data streaming is defined below. These settings must match the GCS being used. Recommend using 224.10.10.10 and 14550 for typical scenarios.

ATAK CoT Setup

These settings are used for the ATAK Cursor-on-target generator. The mulicast address should match the CoT multicast address defined by the TAK network administrator. To disabled the ATAK CoT messages, set period to disabled.

FMU Setup

Vehicle controller settings. These settings must match the FMU firmware settings. Changing the system ID here does not update the system ID, this value must match the FMU firware settings. Do not change unless you know what you are doing.
